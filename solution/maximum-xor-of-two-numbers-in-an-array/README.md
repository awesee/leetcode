## 421. Maximum XOR of Two Numbers in an Array (Medium)

<p>Find the maximum result of a<sub>i</sub> XOR a<sub>j</sub>, where 0 &le; <i>i</i>, <i>j</i> &lt; <i>n</i>.</p>

<p>Could you do this in O(<i>n</i>) runtime?</p>
